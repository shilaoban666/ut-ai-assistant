# Tape -- AI UT assistant
**AI 驱动的智能单元测试生成 & 覆盖率提升插件（IntelliJ IDEA Plugin）**

Tape -- AI UT assistant 是一个面向 Java 开发者的 IntelliJ 插件，集成 GPT / DeepSeek 等大模型，自动分析 JaCoCo 覆盖率、补全未覆盖的方法、插入 TODO 提醒、生成 JUnit 单元测试，并可在一次对话后自动触发测试运行与覆盖率报告刷新。

本插件通过 AI 让“写单测”变成一次对话，显著提升研发效率，是构建高质量代码与持续交付的利器。

---

## ✨ **核心功能**
### 🔍 1. AI 聊天式单元测试生成
- 支持 GPT、GMINI、DeepSeek 等模型
- 用户通过自然语言对话即可生成高质量 JUnit5 单测
- 支持 Mock（Mockito）、边界情况、异常路径自动补全
- 自动美化 Markdown + 代码渲染 + 气泡式聊天 UI

### 📊 2. 自动解析 JaCoCo 覆盖率
- 自动读取 `jacoco.exec` / Jacoco XML
- 展示包/类/方法级覆盖率
- UI 支持排序、搜索、跳转源代码

### 🚨 3. 自动识别低覆盖方法
- 自动定位每个类中覆盖率低于阈值的方法
- 可视化展示 + 红色警告标识
- 自动生成测试补全建议

### 📝 4. 自动插入
- 对低覆盖率方法自动插入单元测试
- 对边界情况自动插入单元测试

### 🤖 5. 自动补全未覆盖方法
- AI 自动生成未覆盖方法的测试样例

- 自动写入 src/test/java

- 自动组织 package & 文件结构

### ▶️ 6. 自动执行测试 & 刷新覆盖率

- 一次对话结束后自动运行测试

- 自动刷新最新 Jacoco 报告

- 整个过程可视化呈现
